OpenGL Framework

Maak een proof-of-concept spel met behulp van je eigen 2D OpenGL framework. Je basis uitgangspunt is bijgeleverde code.

Deze code is een 'opgeschoonde' (en 2D gemaakte) versie van Tutorial 6 op http://www.opengl-tutorial.org.

Wil je een 3D framework maken, Tutorial 8-10 zijn dan goede uitgangspunten.

Bij voorkeur heeft je framework unieke eigenschappen, speciaal toegespitst op jouw type spel. Denk bijvoorbeeld aan:

  • Lighting
  • Basic collision
  • Physics (Box2D)
  • Particle system
  • Audio
  • Multiplayer / networking (sockets)
  • File IO (save/load levels/settings) -> TXT? XML? JSON?
  • Communicate with hardware (serial port / USB / MIDI)?
  • Custom controllers? Kinect? Oculus Rift?
  • (My)SQL connection
  • Draw lines or pixels
  • Tools (level-editors / importers-exporters)
  • Fonts
  • Command line arguments (int main(int argc, char * argv[]) {})
  • Read/write settings to/from file
  • ...

Opleverspecificaties
